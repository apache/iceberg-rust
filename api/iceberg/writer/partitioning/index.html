Partitioning writers for handling partitioned Iceberg tables.

This module provides two strategies for writing to partitioned tables:

  • FanoutWriter: Handles unsorted data by maintaining multiple active writers
  • ClusteredWriter: Optimized for pre-sorted data with single active writer
