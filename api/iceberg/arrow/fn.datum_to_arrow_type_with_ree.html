pub fn datum_to_arrow_type_with_ree(datum: &Datum) -> DataType
Expand description

Converts a Datum (Iceberg type + primitive literal) to its corresponding Arrow DataType with Run-End Encoding (REE).

This function is used for constant fields in record batches, where all values are the same. Run-End Encoding provides efficient storage for such constant columns.
